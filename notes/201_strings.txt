стандартная библиотека string.h отвечает за манипуляции над строками, объединение, копирование, сравнивание и тд.

массив строк
каждая строка массив, поэтому это массив массивов
каждая строка не дляинее 79
char tracks[][80]

strchr()    ищет вхождение символа в строке
strcmp()    сравнивает две строки
strstr()    ищет вхождение одной строки в другую
strcpy()    копирует одну строку в другую
strlen()    опредедяет длину строки
strcat()    соединяет две строки

узнать больше в консоли linux:
man string
man strstrq
------------------------------------------
Используйте wchar_t и связанные функции (wprintf, wcslen и т. д.) для лучшей поддержки символов, не входящих в набор ASCII.
```c
#include <wchar.h>
#include <locale.h>

int main() {
    setlocale(LC_ALL, "");  // Set locale for proper character handling
    wchar_t str[] = L"Привет, мир!";
    wprintf(L"%ls\n", str);  // Use wide character printing
    return 0;
}
```
